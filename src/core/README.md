This directory contains cross-runtime core modules used across the extension:

- `constants.js`: shared extension constants, identifiers, and runtime guards.
- `functions.js`: runtime-independent utility helpers and messaging helpers.
- `tab.js`: `Tab` domain model and Salesforce URL/tab transformations.
- `tabContainer.js`: tab collection model plus persistence/sort/import/export behaviors.
- `translator.js`: locale message lookup and interpolation utilities.
