This directory contains all files used to build and run the extension.

- [action](./action/README.md): browser action pages, popup flows, and related UI assets.
- [background](./background/README.md): background runtime behavior and context-menu wiring.
- [components](./components/README.md): reusable custom elements shared across popup and settings pages.
- [core](./core/README.md): shared domain/runtime modules used across all extension surfaces.
- [entrypoints](./entrypoints/README.md): explicit bundle roots used by the build pipeline.
- [generated](./generated/README.md): generated build artifacts.
- [salesforce](./salesforce/README.md): content-script feature logic for Salesforce pages.
- [settings](./settings/README.md): the extension options page exposed through the manifest.
